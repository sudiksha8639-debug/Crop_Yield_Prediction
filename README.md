# 🌾 Crop Yield Prediction using Linear Regression

> A beginner-friendly Machine Learning mini project that predicts crop yield using agricultural and environmental factors.

---

## 📌 Overview

This project demonstrates the complete workflow of a supervised Machine Learning regression model using **Linear Regression**. It includes data preprocessing, exploratory data analysis (EDA), feature encoding, model training, and performance evaluation on an agriculture dataset.

---

## 🚀 Project Workflow

- 📂 Load and explore the dataset
- 🧹 Handle missing values
- 📊 Perform Exploratory Data Analysis (EDA)
- 🔄 Encode categorical features
- 📈 Analyze feature correlations
- ✂️ Split data into training and testing sets
- 🤖 Train a Linear Regression model
- 📉 Evaluate model performance
